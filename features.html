AllTab v1.0.3

Manage all your tabs in a single place.

Smart Tab Management

  • One-Click Save: Save all tabs in current window or all windows at once.
  • Flexible Saving: Save just this tab, tabs to the left, tabs to the right, or everything except this tab.
  • The Great Reset: Click the dashboard logo to save and close all windows instantly, freeing maximum resources.
  • Context Menus: Right-click on any page to access quick save options.
  • Auto-Send to Dashboard: Automatically save and close inactive browser tabs after a configurable timeout.
  • Stay in Current Window: Option to remain in your current window after saving tabs.
  • Save Browser Tab Group: Save a Chrome native tab group directly along with its name and color.
  • Restore Entire Group: Open all tabs in a saved group at once with a single click.

Visual Organization

  • Automatic Grouping: Tabs saved together are automatically grouped by session.
  • Group Renaming: Give custom names to your groups for easier identification.
  • Drag and Drop: Reorder tabs and groups with full drag-and-drop support.
  • Tags: Assign tags to groups for visual categorization.
  • Search: Instantly search across all your saved tabs and group names.
  • Date Filter: Filter groups by the date they were created.
  • Tag Filter: Show only groups matching a specific tag.
  • Inter-Group Creation: Hover between groups and click the + button to insert a new empty group at that position.
  • Scroll to Top: Quick button to return to the top of your tab list.

Performance and Efficiency

  • RAM Monitor: Real-time display of System and Chrome memory usage in the popup.
  • Tab Hibernation: Suspend inactive tabs to free memory without closing them.
  • Auto-Hibernate: Set timers (1 min to 2 hours) to automatically hibernate tabs.
  • Hibernation Whitelists: Exclude specific sites (like music players) from hibernation.
  • Hibernate from Context Menu: Hibernate this tab, tabs to the left, tabs to the right, all except this, or wake all hibernated tabs.
  • Sleeping Page Controls: Sleeping tabs can be set to wake on page reload or when you visit them.
  • Duplicate Cleanup: Automatically identify and remove duplicate tabs.
  • Automatic Tab Cleanup: Automatically remove old, unused tabs after a set period (days, weeks, or months).
  • Cleanup Exclusions: Whitelist domains to protect them from automatic cleanup.

Data Safety and Privacy

  • Offline First: All data stays on your device. No external analytics or tracking.
  • Trash Bin: Deleted items are kept safely for 30 days before permanent removal.
  • Undo System: Accidentally deleted a tab? You have a 20-second window to bring it back.
  • Group Locking (Pin): Lock important groups to protect them from accidental deletion or cleanup.
  • Backup (JSON): Export your entire collection to a JSON file for safekeeping.
  • Restore (JSON): Import a JSON backup with options to merge or replace, handle duplicates, and select specific groups.

Import and Export Options

  • Import from JSON: Restore from a full backup file with drag-and-drop support.
  • Import from Text: Paste a list of URLs to create a new group.
  • Import from Chrome Bookmarks: Pull in tabs from your existing Chrome bookmarks.
  • Export as JSON: Full backup for restoration.
  • Export as HTML: Generate a styled, shareable HTML file with theme options (Light, Dark, Ocean, Forest, Sunset, Lavender, Midnight).
  • Export as HTML Lite: A minimal, lightweight HTML export.
  • Export as Text (URLs): Plain text list of all your saved URLs.
  • Export as CSV: Spreadsheet-compatible format for external tools.
  • Print / PDF: Print your tab list or save it as a PDF.

View Modes and Customization

  • Standard Dashboard: The classic full-page view for managing everything.
  • Sidebar Mode: A compact side-panel view for quick access while you browse.
  • Wide Mode: An optimized layout for large monitors.
  • Multi-User Profiles: Create separate profiles to keep tab collections distinct.
  • Theme Toggle: Switch between light and dark mode.
  • URL Display Options: Choose to show full URL, domain only, shortened, or hidden.
  • Pagination: Configure how many tabs to display per page.
  • 13 Languages: English, Spanish, French, Portuguese, German, Chinese, Japanese, Russian, Hindi, Arabic, Bengali, Urdu, Indonesian.
  • Open on Startup: Optionally open AllTab automatically when Chrome starts.
  • One-Click Save: Configure the extension icon to save immediately on click.
  • Delete on Open: Optionally remove a tab from the list when you restore it.
  • Switch to Tab: Optionally focus a restored tab immediately.
  • Open Sleeping: Optionally open restored tabs in a hibernated state to save memory.
